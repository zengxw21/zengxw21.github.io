Reinforcement Learning Notes

1. 贝尔曼公式

考虑这样一个状态转移，agent 在状态 S 采取动作 A，获得 reward R 并转移到下一个状态：
$S_t\xrightarrow{A_t}R_{t+1},S_{t+1}\xrightarrow{A_{t+1}}R_{t+2},S_{t+2}\xrightarrow{A_{t+2}}R_{t+3},\ldots$

我们计算整条路径上 reward 之和也就是 return
$\begin{aligned}
G_{t} & =R_{t+1}+\gamma R_{t+2}+\gamma^{2}R_{t+3}+\ldots, \
& =R_{t+1}+\gamma(R_{t+2}+\gamma R_{t+3}+\ldots), \
& =R_{t+1}+\gamma G_{t+1},
\end{aligned}$

那么如何计算从 s 开始，采用策略$\pi$获得的 return 的期望，也就是 state value 呢？它等于
$\begin{aligned} v_{\pi}(s) & =\mathbb{E}[G_t|S_t=s] \ & =\mathbb{E}[R_{t+1}+\gamma G_{t+1}|S_{t}=s] \ & =\mathbb{E}[R_{t+1}|S_t=s]+\gamma\mathbb{E}[G_{t+1}|S_t=s] \end{aligned}$

分别计算这两项，第一项：
$\begin{aligned} \mathbb{E}[R_{t+1}|S_{t}=s] & \begin{aligned} =\sum_a\pi(a|s)\mathbb{E}[R_{t+1}|S_t=s,A_t=a] \end{aligned} \ & =\sum_a\pi(a|s)\sum_rp(r|s,a)r \end{aligned}$

其中$\pi(a|s)$ 表示在 policy $\pi$下，在状态 s 采取动作 a 的概率；$p(r|s,a)$ 表示在状态 s 采取动作 a 后，获得奖励 reward 的概率。那么上面的式子其实就是求第一步 reward 的期望。

接下来看第二项，
$\begin{aligned} \mathbb{E}[G_{t+1}|S_{t}=s] & \begin{aligned} =\sum_{s^{\prime}}\mathbb{E}[G_{t+1}|S_t=s,S_{t+1}=s^{\prime}]p(s^{\prime}|s) \end{aligned} \ & =\sum_{s^{\prime}}\mathbb{E}[G_{t+1}|S_{t+1}=s^{\prime}]p(s^{\prime}|s) \ & =\sum_{s^{\prime}}v_\pi(s^{\prime})p(s^{\prime}|s) \ & =\sum_{s^{\prime}}v_\pi(s^{\prime})\sum_ap(s^{\prime}|s,a)\pi(a|s) \end{aligned}$

其中 $\mathbb{E}[G_{t+1}|S_t=s,S_{t+1}=s^{\prime}] = \mathbb{E}[G_{t+1}|S_{t+1}=s^{\prime}]$ 的原因是，整个过程是 Markov 决策，也就是与历史状态无关。而 $v_\pi(s^{\prime})$ 指的是从 $s^{\prime}$ 开始，能得到的 return 的期望。
这一项求的是从下一个状态开始能得到的 return 的期望。

回头看开始的式子，提取出公共项，就有
$ v_{\pi}(s)=\sum_a\pi(a|s)\left[\sum_rp(r|s,a)r+\gamma\sum_{s^{\prime}}p(s^{\prime}|s,a)v_\pi(s^{\prime})\right],\quad\forall s\in\mathcal{S}.$

注意这里的集合符号，代表这是一组等式，每个 state 都有一个方程。联立这个方程组，我们就能求出每个状态的 state value，进而评估这个策略的好坏。

接下来看 action value 的定义：
$q_\pi(s,a)=\mathbb{E}[G_t|S_t=s,A_t=a]$

它指的是在状态 s，采取动作 a 后，获得 return 的期望。那么我们不难得出，state value 其实就是 action value 的加权平均。
$v_\pi(s)=\sum_a\pi(a|s)q_\pi(s,a)$

将其与上面得到的式子作比较，发现
$q_\pi(s,a)=\sum_rp(r|s,a)r+\gamma\sum_{s^{\prime}}p(s^{\prime}|s,a)v_\pi(s^{\prime})$

这说明 state value 和 action value 其实可以互相求解。
